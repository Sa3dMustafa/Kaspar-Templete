# Kasper Template Website 🎨

A modern and responsive multi-section landing page built باستخدام HTML و CSS.

## 🚀 Features

* Fully Responsive Design 📱
* Animated Image Slider 🎞️
* Modern UI Sections (Services, Portfolio, Pricing, Contact)
* Smooth Scrolling Navigation
* Clean and Structured Code

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox + Grid)
* Font Awesome Icons
* Google Fonts

## 📸 Sections Included

* Header & Navigation
* Landing Section (Slider)
* Services
* Portfolio Gallery
* Video Section
* About
* Stats
* Testimonials & Skills
* Pricing Plans
* Contact Form
* Footer

## 🎯 What I Learned

* Building full multi-section layouts
* Working with Flexbox & Grid
* Creating CSS animations (slider)
* Improving UI/UX design skills
